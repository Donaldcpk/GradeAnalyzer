# 成績數據分析系統

這是一個用於分析學生成績的系統，提供班級、年級和個人成績的預測分析。

## 功能特點

- 支援班級成績預測
- 支援年級成績預測
- 支援個人成績預測
- 互動式圖表顯示
- 支援示範數據測試

## 使用方法

1. 啟動系統後，您可以：
   - 選擇使用示範數據來測試系統功能
   - 上傳實際的成績 Excel 檔案進行分析

2. Excel 檔案格式要求：
   - 必要欄位：
     - *School Year（學年度）
     - *Class Level（年級）
     - *Class（班級）
     - *Class Number（座號）
     - *Student Name（學生姓名）
   - 成績欄位格式：T{term}A{assessment}_{Subject}_{Language}_Score
     - 例如：T1A1_Math_C_Score（第一學期第一次評量數學中文卷）

3. 系統將自動生成：
   - 班級預測圖表
   - 年級預測圖表
   - 個人成績預測

## 注意事項

- 預測功能需要至少兩次以上的考試成績才能進行
- 滑鼠移到圖表上的數據點可查看詳細資訊
